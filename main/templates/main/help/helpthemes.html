Themes Guide



By default, Daily Note is presented in a pastel theme to its users. However, this theme can be changed depending on the mood of the user. The change in theme mainly covers the background of the page and the color of the navigation bar only.

Changing the Theme:

1. On the Navigation Bar on the top of the website, click the section on the top right corner that contains the user's name.

2. After clicking Hi __________, choose between the pastel theme, neutral theme, and bright theme.

3. The user will be redirected to a dashboard to see the changes. Afterwards, the user can now proceed again to using web application.
