Calendar Guide



Daily Note provides also provides its users with a calendar view of the events and tasks that need to be accomplished at hand. Through the calendar view, users can add events/tasks that they need to take note of.

Adding an Event/Task

1. On the Navigation Bar on the top of the website, click the Calendar section.

2. On the top right hand corner, click the Add Event button.

3. On the modal window that appears, choose whether a task or an event will be taken note of.

4. Enter the event title, description.

5. Enter the start date and end date by clicking the interactive calendar on the side.

> Note that tasks will be displayed in the calendar according to its start date.

6. Click the Save button to save the event/task, or the Close button to abort.
